# **App Name**: Mind Bloom

## Core Features:

- Game Selection: Game selection screen: Allows users to choose from available brain exercise games.
- Memory Match: Memory Match Game: Presents a card matching game to improve memory and focus.
- Logic Puzzles: Logic Puzzle Game: Offers a variety of simple logic puzzles, such as number sequences or spatial reasoning.
- Tracked Progress: Progress Tracking:  Records the number of successfully completed games of each type to provide metrics.

## Style Guidelines:

- Primary color: Calming green (#8FBC8F) to evoke a sense of tranquility and focus.
- Background color: Light gray (#D3D3D3), providing a neutral backdrop that doesn't distract from the games.
- Accent color: Soft yellow (#E8D882), used to highlight interactive elements.
- Body and headline font: 'PT Sans' sans-serif, for a modern, legible look.
- Simple, geometric icons that are easy to understand at a glance.
- Clean and spacious layouts to prevent cognitive overload.
- Subtle transitions and animations for a smooth user experience.
